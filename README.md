# latex-itemizer

For latex-itemizer

## Usage

| Command Name                   | Key Binding               |
|--------------------------------|---------------------------|
| latex-itemizer:newline-item    | ```Alt         + Enter``` |
| latex-itemizer:newline-itemize | ```Alt + Shift + Enter``` |
